# rez-avalon

This package builds avalon-core from MoonShineVFX/avalon-core production branch.

### Usage

```shell script
$ rez-env ozark avalon
$ cd path/to/project
$ inventory --init
$ echo "Edit .toml"
$ inventory --save
$ party --init
$ echo "Edit package.py"
$ party --at avalon
$ echo "Launch Allzpark"
$ park
```
